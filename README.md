# 2048
This is a simple implementation of the classic 2048 game built using HTML, CSS, and JavaScript. The game is played on a 4x4 grid where the objective is to merge tiles with the same numbers to reach the tile "2048".

## Features
Interactive gameplay: Use arrow keys or on-screen controls to play.
Score tracking: Displays your current score during gameplay.
Win and Game Over screens: Alerts you when you've won or lost.
Responsive design: Works seamlessly on desktop and mobile browsers.
## How to Play
Start the game by clicking the "Start Game" button.
Use the arrow keys or the on-screen buttons to move the tiles:
Tiles with the same number will merge when they collide.
Each move adds a new tile with a value of 2 or 4 in a random empty spot.
Keep merging tiles until you create a tile with the number 2048 to win!
The game ends if there are no valid moves left.
